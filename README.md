# Vansh Mirani Portfolio

A modern personal portfolio website built to showcase my projects, technical skills, education, certifications, training experience, and contact information.

This portfolio is designed for internship applications and professional networking, with a clean dark theme, responsive layout, interactive project cards, and a recruiter-friendly structure.

## Getting Started

Follow these steps to run the project locally.

### 1. Clone the Repository

```bash
git clone https://github.com/VanshMirani/Vansh_Mirani_Portfolio.git
```

### 2. Navigate to the Project Folder

```bash
cd Vansh_Mirani_Portfolio
```

### 3. Install Dependencies

```bash
npm install
```

### 4. Start the Development Server

```bash
npm run dev
```

The project will run locally at:

```text
http://localhost:5173
```

---

## Build for Production

To create a production build, run:

```bash
npm run build
```

To preview the production build locally:

```bash
npm run preview
```
